# 要求定義書：ゲーム時間選択機能

## 概要

ゲーム開始前にプレイヤーがゲームの制限時間を選択できる機能を追加する。

## 背景・動機

現在、ゲーム時間は `gameConfig.ts` の `DURATION_SECONDS = 30` として固定されている。
短時間でサクッと遊びたいユーザーや、じっくり競いたいユーザーに対応するため、複数の時間オプションを提供する。

## ユーザーストーリー

> **As a** プレイヤー
> **I want to** ゲーム開始前に制限時間を 10秒・30秒・60秒 から選択したい
> **So that** プレイスタイルや目的に合わせてゲームを楽しめる

## 受け入れ条件

1. タイトル画面に時間選択 UI が表示される
2. 選択できる時間は **10秒 / 30秒 / 60秒** の3種類
3. デフォルト選択は **30秒**
4. ゲーム中のタイマーは選択した時間からカウントダウンする
5. リザルト画面の KPM 計算は選択した時間を元に正しく算出される
6. 「もう一度対戦」で再スタートしても、同じ選択時間が引き継がれる
7. タイトル画面に戻ったあと、時間選択は前回の値を保持する

## 制約事項

- タイトル画面の既存レイアウト（音源選択・サウンドボタン）を大きく崩さない
- 時間選択は対戦設定の一部として扱い、ゲーム進行中は変更できない
