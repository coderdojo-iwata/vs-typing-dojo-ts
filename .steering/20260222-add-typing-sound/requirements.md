# 要求内容: キー入力サウンドエフェクトの追加

## 変更・追加する機能の説明

タイピングゲームのキー入力時に、ゲーム風の電子音を再生する機能を追加する。
入力結果（正解・不正解）に応じて異なる音を鳴らし、プレイヤーへの即時フィードバックを強化する。

## ユーザーストーリー

- タイピング中に音が鳴ることで、画面を見なくても入力の正否がわかる
- ゲーム風の電子音によって、プレイのテンポ感と没入感が高まる
- 音が不要な環境（図書館・深夜など）でも快適にプレイできる

## 受け入れ条件

- [ ] 正しいキーを入力したとき（部分一致・チャンク完成）に同じ音が鳴る
- [ ] 1課題文をすべて打ち終えたとき（ミスあり完成）に、通常の正解音とは異なる特別な音が鳴る
- [ ] 1課題文をノーミスで打ち終えたとき（ノーミス完成）に、ミスあり完成音とも異なる音が鳴る
- [ ] 誤ったキーを入力したとき（不正解）に、正解とは明確に異なる音が鳴る
- [ ] 音はゲーム風の電子音（Web Audio API で生成）とする
- [ ] 外部の音声ファイルは使用しない
- [ ] プレイヤー1・プレイヤー2 それぞれの入力に対して音が鳴る
- [ ] タイトル画面でサウンドの ON/OFF を切り替えられる
- [ ] サウンド設定は localStorage に保存され、次回起動時も引き継がれる

## 制約事項

- ライブラリの追加は行わない（Web Audio API のみ使用）
- 音量はユーザーの OS/ブラウザの音量設定に依存する（音量コントロールUIは今回追加しない）
- ブラウザの自動再生ポリシーに対応すること（ユーザー操作後に AudioContext を初期化する）
